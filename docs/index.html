Online Experiments

This website contains the following 10 online tests. These were made to study the passive action recognition, the active action recognition and the visual perspective taking in humans:

  1. Proactive Action Recognition with No Movements, Very Unclear Images, No T-Poses, No Obstacles;
  2. Proactive Action Recognition with Self-Controlled Movements, Slightly Unclear Images, No T-Poses, No Obstacles.
  3. Proactive Action Recognition with Self-Controlled Movements, Very Unclear Images, No T-Poses, No Obstacles.
  4. Proactive Action Recognition with Self-Controlled Movements, Slightly Unclear Images, T-Poses, No Obstacles.
  5. Proactive Action Recognition with Random Movements, Slightly Unclear Images, No T-Poses, No Obstacles.
  6. Proactive Action Recognition with Random Movements, Very Unclear Images, No T-Poses, No Obstacles.
  7. Proactive Action Recognition with Random Movements, Slightly Unclear Images, T-Poses, No Obstacles.
  8. Proactive Action Recognition with Self-Controlled Movements, Very Unclear Images, No T-Poses, Obstacles.
  9. View Theta Recognition.
  10. View Phi Recognition.
